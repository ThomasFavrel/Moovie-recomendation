# Movie Recommendation

This repo shows two different methhods of movie recommendation systems for the [MovieLens 1M dataset](https://grouplens.org/datasets/movielens/1m/). The dataset contain 1,000,209 anonymous ratings of approximately 3,900 movies made by 6,040 MovieLens users who joined MovieLens in 2000.

## Requirements

* [Python 2.7](https://www.python.org/download/releases/2.7/) or [Python 3.6](https://www.python.org/downloads/release/python-360/)
* [Jupyter Notebook](http://jupyter.org/)
